Extensions to the OWL-Time Ontology - Eras

OWL-Time [[owl-time]] is an ontology for temporal entities and relations between them. OWL-Time defines simple temporal entities (intervals and instants). This note adds one new class time:Era, two properties time:era and time:reignOf, and an instance time:CommonEra to allow for the description of eras.

The namespace for OWL-Time terms is http://www.w3.org/2006/time#

The suggested prefix for the OWL-Time namespace is time

The OWL-Time ontology is available here.

The OWL-Time ontology with these extensions is available here.

New classes and properties are introduced in this extension to OWL-Time. The new elements concern temporal entities which cannot be expressed in OWL-Time. The ontology document imports OWL-Time, and adds the new elements and axioms. IRIs denoting the new elements are placed in the OWL-Time namespace since it is expected that they could be added to the normative ontology in a future revision of OWL-Time.

Motivation and background

OWL-Time is an ontology for describing temporal information. As part of its mission it implements ways to encode dates in the Gregorian calendar.

This extension answers a need by Digital Humanists to encode dates in other types of calendar using different eras.

Vocabulary specification

Classes

Era

Class: time:Era
IRI: http://www.w3.org/2006/time#Era
Definition: A linear (non-cyclic) year numbering system. No assumption is made with regards to the details of how years are numbered in the era, such as a year 0, or years with negative indexes. Any year numbering system can be an instance of this class even if the year definition doesn't match the one of the Gregorian calendar. For instance Roman regnal years or Chinese eras can be encoded as instances of time:Era.
Instance of: owl:Class

Properties

era

Property: time:era
IRI: http://www.w3.org/2006/time#era
Definition: Era indication
Instance of: owl:ObjectProperty
Domain: time:GeneralDateTimeDescription
Range: time:Era

era

Property: time:reignOf
IRI: http://www.w3.org/2006/time#era
Definition: Indicates the person whose reign is associated with the era. Most eras are assciated with one reign (ex: Roman regnal years), but sometimes several eras are associated with one reign (ex: later Chinese eras). This property can be used in both cases. No range is indicated so that the associated person can have any relevant type.
Instance of: owl:ObjectProperty
Domain: time:Era

Individuals

time:CommonEra

The Era of the modern gregorian calendar. It is defined so that a general year in a date object associated with this era is roughly equivalent to a gYear. This specification doesn't say anything about the general year being a year of the gregorian calendar, so any type of year can be used. For example, a date in the Chinese calendar may be given using a year indicated in the Common Era, even though the Chinese year starts and ends after the one of the Gregorian calendar.

CommonEra

Class Individual
time:Era time:CommonEra

Examples

Date in a Roman calendar

The 4th year in the reign of Diocletian could be noted as follows: 

ex:DiocletianRegnalYears
  a time:Era ;
  time:reignOf ex:Diocletian ;
.
ex:Year4OfDiocletiansReign a time:GeneralDateTimeDescription ;
  time:year "0004"^^time:generalYear ;
  time:era ex:DiocletianRegnalYears ;
